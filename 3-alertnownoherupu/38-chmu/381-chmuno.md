# 3.8.1 チームの追加

1. 画面上部&#x306E;**【チーム作成】**&#x30DC;タンをクリックします。



2. チーム追加ポップアップに表示される入力項目は以下の通りです。



| 項目       | 説明                           | 備考   |
| -------- | ---------------------------- | ---- |
| 名前       | チーム名を入力します。                  | 入力必須 |
| エスカレーション | チームに追加するエスカレーションを選択します。      | 選択事項 |
| チーム公開設定  | チームの公開・非公開を選択します。デフォルトは全体公開。 | 入力必須 |
| 説明       | チームに関する説明を入力します。             | 選択事項 |

3. **【確認】**&#x30DC;タンをクリックして、チームを追加します。
